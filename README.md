#Mudo Pet Salinas

Es una tienda online que vende productos para mascotas domesticas y exoticas.

##Descripcion

Esta pagina web fue creada con el objetivo  de vender en modo online la mejor calidad de alimentos y productos para los animales domesticos y exoticos.

##Tecnología usada

HTML, CSS, SASS, BOOTSTRAP Y FONTWESOME.
